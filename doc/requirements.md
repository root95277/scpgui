# SCP GUI - 需求文档

## 1. 项目概述

SCP GUI 是一款基于 Python 的图形化文件传输工具，采用双面板设计（类似 Midnight Commander），通过 SSH/SFTP 协议实现本地与远程服务器之间的文件上传和下载。

## 2. 技术栈

| 组件     | 技术选型              | 说明                     |
| -------- | --------------------- | ------------------------ |
| 语言     | Python 3.10+          | 主开发语言               |
| GUI 框架 | PySide6 (Qt6)         | 跨平台图形界面           |
| SSH/SFTP | paramiko              | SSH 连接与文件传输       |
| 打包     | PyInstaller (可选)    | 打包为 macOS .app        |

## 3. 界面设计

### 3.1 整体布局

```
+------------------------------------------------------------------+
|  工具栏: [连接] [断开] [刷新] [设置]                               |
+-------------------------------+----------------------------------+
|  远程服务器 (左面板)            |  本地文件 (右面板)                |
|  路径: /home/user/             |  路径: /Users/jack/Downloads/    |
|  [..] (上级目录)               |  [..] (上级目录)                 |
|  [DIR] documents/     2026-04  |  [DIR] projects/       2026-04  |
|  [DIR] downloads/     2026-03  |  [DIR] images/         2026-03  |
|  [FILE] readme.txt    1.2 KB   |  [FILE] notes.txt      3.4 KB  |
|  [FILE] data.csv      45 MB    |  [FILE] report.pdf     2.1 MB  |
+-------------------------------+----------------------------------+
|  状态栏: 已连接 user@192.168.1.100 | 传输进度条                    |
+------------------------------------------------------------------+
```

### 3.2 面板说明

- **左面板（远程）**: 显示远程服务器的文件和目录列表
- **右面板（本地）**: 显示本地文件系统的文件和目录列表
- **每个面板包含**:
  - 当前路径显示/编辑栏
  - 文件列表（列: 图标/类型、文件名、大小、修改日期）
  - 支持双击进入目录、双击 `..` 返回上级
  - 支持排序（按名称、大小、日期）

## 4. 功能需求

### 4.1 SSH 连接管理

- **读取 ~/.ssh/config**: 启动时自动解析 `~/.ssh/config`，列出已配置的服务器供用户选择
- **手动输入连接**: 支持手动填写连接信息：
  - 主机地址 (Host)
  - 端口号 (Port, 默认 22)
  - 用户名 (Username)
  - 认证方式：密码 / 私钥文件
- **连接状态**: 状态栏显示当前连接状态（未连接/连接中/已连接/连接失败）
- **断开连接**: 支持手动断开当前连接

### 4.2 文件浏览

- **远程文件浏览**: 通过 SFTP 列出远程目录内容
- **本地文件浏览**: 列出本地文件系统目录内容
- **显示信息**: 文件名、文件大小（人类可读格式）、修改时间、文件类型图标
- **目录导航**: 双击目录进入，双击 `..` 返回上级，路径栏可直接编辑跳转
- **隐藏文件**: 支持显示/隐藏以 `.` 开头的隐藏文件（默认隐藏）
- **刷新**: 手动刷新当前目录列表

### 4.3 文件传输（核心功能）

- **拖拽上传**: 从右面板（本地）拖拽文件/目录到左面板（远程）→ 上传
- **拖拽下载**: 从左面板（远程）拖拽文件/目录到右面板（本地）→ 下载
- **目录递归传输**: 拖拽目录时递归传输整个目录树
- **传输进度**: 显示当前传输文件名、进度百分比、传输速度
- **覆盖确认**: 目标路径已存在同名文件时弹出确认对话框（覆盖/跳过/取消）
- **后台传输**: 传输过程不阻塞 UI，使用独立线程处理

### 4.4 基本文件操作

- **新建文件夹**: 在当前目录（本地或远程）新建文件夹
- **删除**: 删除选中的文件/目录（带确认对话框）
- **重命名**: 重命名选中的文件/目录
- **右键菜单**: 右键弹出操作菜单（新建文件夹、删除、重命名、刷新）

## 5. 非功能需求

- **目标平台**: macOS（兼容 Apple Silicon 和 Intel）
- **性能**: 大目录（1000+ 文件）列表加载不卡顿，使用异步加载
- **错误处理**: 网络中断、权限不足、磁盘满等异常情况给出清晰提示
- **安全**: 不明文存储密码，私钥路径仅存储路径引用
